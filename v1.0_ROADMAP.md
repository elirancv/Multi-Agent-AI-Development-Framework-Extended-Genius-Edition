# v1.0 Roadmap

**Target Release**: Q2 2025
**Status**: Planning

## Goal

v1.0 will be the first **stable release** with:
- Feature freeze (bug fixes only)
- Comprehensive test coverage (>90%)
- Complete API documentation
- Performance benchmarks
- Security audit
- Migration guide

## Core Features

### 1. API Documentation Generator

**Priority**: High
**Status**: Planned

- Generate API docs from docstrings
- Output to `docs/api/`
- Auto-generate on release
- Searchable and browsable

**Tools**: pdoc or pydoc-markdown

### 2. Pipelines Gallery

**Priority**: Medium
**Status**: Planned

- Example pipelines with screenshots
- Use case documentation
- Best practices guide
- Visual pipeline diagrams

**Location**: `docs/pipelines/`

### 3. Council Presets

**Priority**: Medium
**Status**: Planned

- YAML shortcuts for common configurations
- Presets: MVP, Production, Research
- CLI flag: `--preset <name>`
- Easy switching between modes

### 4. Artifact Retention Policy

**Priority**: Medium
**Status**: Planned

- Automatic cleanup
- Size limits per run
- Time-based retention
- Keep latest N runs
- CLI command: `cleanup-artifacts`

### 5. Performance Benchmarks

**Priority**: High
**Status**: Planned

- Baseline performance metrics
- Regression detection
- Performance test suite
- CI integration

## Quality Improvements

### Test Coverage

- **Current**: >80%
- **Target**: >90%
- **Focus Areas**:
  - Edge cases
  - Error handling
  - Integration tests
  - Performance tests

### Documentation

- Complete API reference
- Migration guide from v0.9.x
- Troubleshooting guide
- Performance tuning guide

### Security

- Security audit
- Dependency vulnerability scanning
- Secure defaults
- Security best practices guide

## Breaking Changes

**None planned** - v1.0 will maintain backward compatibility with v0.9.x.
