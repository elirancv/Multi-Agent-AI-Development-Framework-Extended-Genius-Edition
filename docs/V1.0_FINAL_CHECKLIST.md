## ✅ Pre-Release Actions

1. **Run Doctor**
   ```bash
   python scripts/doctor.py --verbose
   ```
   Should pass all checks (exit code 0)

2. **Audit Dependencies**
   ```bash
   python scripts/deps_audit.py
   ```
   Review `docs/deps-audit.md` for any blocked licenses

3. **Create Performance Baseline**
   ```bash
   python scripts/perf_baseline.py
   ```
   Baseline saved to `docs/benchmarks/`

4. **Verify Version**
   ```bash
   python cli.py --version
   ```
   Should output: `1.0.0`

5. **Test Doctor Command via CLI**
   ```bash
   python cli.py doctor
   ```
